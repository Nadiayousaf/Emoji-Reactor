# 😃 Emoji Reactor

An interactive, responsive web application that allows users to react to content using animated emojis. Built entirely with vanilla web technologies.
### lIVE repo;      https://nadiayousaf.github.io/Emoji-Reactor/
## ✨ Features
* **Interactive UI:** Smooth hover and click animations for emojis.
* **Responsive Design:** Works flawlessly on mobile, tablet, and desktop screens.
* **Lightweight:** Fast loading times using native HTML, CSS, and JavaScript.

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for the layout.
* **CSS3:** Custom styling, layout (Flexbox/Grid), and transition animations.
* **JavaScript (ES6):** State management and dynamic DOM manipulation for reaction counters.

## 🏃 Getting Started

### Prerequisites
You only need a modern web browser (like Chrome, Firefox, or Safari) to run this project.

### Running Locally
1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Open the project folder:
   ```bash
   cd emoji-reactor
   ```
3. Open `index.html` directly in your web browser to view the application.
